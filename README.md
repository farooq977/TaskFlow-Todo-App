# TaskFlow Todo App

A modern and responsive task management application designed to help users organize daily activities efficiently. The application features task management, Dark/Light mode, task ratings, and browser-based Local Storage for persistent data without requiring a backend.

---

## ✨ Features

- ✅ Create, edit, and delete tasks
- ⭐ Task rating system
- 🌙 Dark Mode
- ☀️ Light Mode
- 💾 Local Storage persistence
- 📱 Responsive design
- ⚡ Fast and lightweight interface
- 🎯 Simple and intuitive user experience

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Local Storage API

The application allows users to:

- Create new tasks
- Edit existing tasks
- Delete completed tasks
- Rate task priority
- Switch between Dark and Light themes
- Automatically save tasks in Local Storage
